# Gym Lens Coach

Gym Lens Coach is an offline browser-based fitness coach built with JavaScript, Vite, and MediaPipe Pose. It uses the webcam to track body landmarks, detect common exercises, count reps or hold time, and show live form feedback while the user exercises.

## Overview

This project explores how real-time pose estimation can be used to create a lightweight workout assistant directly inside the browser. Instead of sending video to a cloud service, the app runs pose tracking and exercise judging on the user's device.

The coach focuses on keeping the important workout information visible on screen while the user moves:

- detected exercise
- rep count
- plank hold time
- joint-angle metrics
- camera framing guidance
- live form feedback

The app is designed for a laptop or desktop webcam and works best when the user's full body is visible in the camera frame.

## How It Works

1. The browser requests webcam access using the Webcam API.
2. Each video frame is sent to MediaPipe Pose.
3. MediaPipe returns body landmarks such as shoulders, elbows, hips, knees, and ankles.
4. The app converts landmarks into measurements such as:
   - elbow angle
   - knee angle
   - hip angle
   - torso angle
   - body line angle
   - crunch ratio
   - hip alignment
   - upper-body motion
5. The pose is grouped into a broad family:
   - idle
   - upright
   - floor
   - transition
6. Exercise scoring logic decides whether the movement is most likely a squat, push-up, plank, or crunch.
7. The selected exercise must stay stable for multiple frames before it is confirmed.
8. Rep counters or hold timers update based on the confirmed exercise.
9. Live feedback tells the user what to fix, such as squat depth, push-up hand position, plank hip alignment, or crunch curl.

## Exercise Logic

### Squats

The app looks for an upright body position with visible knee and hip bending. Standing still is treated as idle, not as a squat.

Measured checks include:

- knee angle
- hip angle
- torso angle
- stance width
- hip movement
- knee tracking

### Push-ups

Push-ups are detected from a supported horizontal body shape plus upper-body movement. This helps separate push-ups from plank, because the starting position can look similar.

Measured checks include:

- elbow bend
- shoulder up/down motion
- hand position under shoulders
- hip alignment
- body straightness

### Plank

Plank is treated as a stable horizontal hold with low motion.

Measured checks include:

- straight body line
- hip height
- elbow lockout
- low upper-body movement
- hold time

### Crunches

Crunches are detected from a tucked floor position and curl motion.

Measured checks include:

- crunch ratio
- knee bend
- hip bend
- shoulder-to-knee distance
- controlled open/closed movement

## Local Setup

### Prerequisites

- Node.js
- npm
- A modern browser such as Chrome or Edge
- Webcam permission enabled

### Install

```powershell
npm install
```

If PowerShell blocks `npm`, use:

```powershell
npm.cmd install
```

## Run the App

Start the Vite development server:

```powershell
npm run dev
```

If needed on Windows PowerShell:

```powershell
npm.cmd run dev
```

Then open the local URL shown in the terminal, usually:

```text
http://localhost:5173/
```

Keep the terminal window open while using the app.

## Build for Production

## Best Camera Setup

- Keep the full body inside the camera frame.
- Step back far enough for shoulders, hips, knees, and ankles to be visible.
- Use a side or slight three-quarter view for push-ups, plank, and crunches.
- Use an upright front or three-quarter view for squats.
- Wear clothes that contrast with the background.
- Allow the model a second to stabilize before starting reps.

## Notes for Public Use

- This project does not require API keys.
- No cloud AI service is required for exercise detection.
- Build output, dependencies, and local artifacts should not be committed to GitHub.
- `node_modules/` and `dist/` are already ignored in `.gitignore`.
- The app is intended as a prototype fitness-coach project, not a medical or professional sports-analysis tool.

## Current Limitations

- Accuracy depends heavily on camera angle, lighting, and full-body visibility.
- Push-ups and plank can look similar in a single frame, so the app uses motion over time to separate them.
- Crunches can be confused with other tucked floor poses if the camera cannot clearly see the body shape.
- The app does not currently save workout history or export reports.
- The app does not use a trained custom exercise classifier; it uses MediaPipe landmarks plus custom measurement rules.

## Disclaimer

Gym Lens Coach is an educational fitness-technology project. It provides computer-vision-based guidance, but it should not replace professional coaching, medical advice, or injury assessment.
